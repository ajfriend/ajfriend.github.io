Regenerating the Figures

All figures in this post are generated from Python code in the code/ directory. The project uses uv to manage dependencies, ensuring reproducibility across different environments and Python versions.

To regenerate the figures:

1cd content/blog/reproducible_plots
2just generate

The Python scripts, dependencies (pyproject.toml and uv.lock), and justfile are all kept in the same folder as this post, making the analysis fully reproducible.
