## bit layout


## H3 cell

|     name    | # of bits | value(s) |
|-------------|-----------|----------|
| reserved    | 1         | 0        |
| mode        | 4         | 1        |
| reserved    | 3         | 0        |
| resolution  | 4         | 0--15    |
| base cell # | 7         | 0--121   |
| digit 1     | 3         | 0--7     |
| $\vdots$    | $\vdots$  | $\vdots$ |
| digit 15    | 3         | 0--7     |

Note that $1 + 4 + 3 + 4 + 7 + 3 \cdot 15 = 64$.
